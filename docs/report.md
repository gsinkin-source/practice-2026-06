# Отчет о ходе разработки проекта

---

## Ход разработки

### Первая стадия: Создание проекта
Проект начался с форка проектной директории из репозитория Мосполитеха.
*   **README обновления:** Изначальные коммиты были направлены на исправление `README.md` файла.

### Вторая стадия: Скелет проекта
В ходе этой стадии была создана архитектура конечного сайта.
*   **Создание скелета:** Был пуш с базовым наполнением HTML/CSS сайта.
*   **Организация файлов:** Файлы были перемещены в специальную папку `/site` с целью улучшения архитектуры проекта.
*   **Реструктуризация файлов:** Были проведены дальнейшие структурные улучшения папки `site`.

### Третья стадия: Добавление новшеств и улучшения сайта с помощью pull- и merge- реквестов.
В ходе проведения проекта был использован подход с ветками разработки.
*   **Модульный подход:** Четыре главных pull-реквеста были одобрены, и происходило слияние главной и тестовой веток.

### Четвертая стадия: UI/UX улучшения и наполнение контентом
Фокус разработки сместился на проработку визуальных элементов и наполнение контентом.
*   **Навигация:** Был исправлен элемент navbar для того, чтобы UX/UI сочетался на всех страницах сайта.
*   **Страница О проекте:** Была закончена страница О Проекте.
*   **Устранение недостатков на главной странице:** Был исправлен CSS на главной странице, и она была заполнена контентом.

### Пятая стадия: Последний коммит
Проект сайта закончен.
